# Obsidian Quick Note

Chrome extension for quick Markdown notes into Obsidian via Local REST API.

## Setup

1. `npm install`
2. `npm run build`
3. Open `chrome://extensions`, enable Developer mode, click Load unpacked, select `dist/`.
